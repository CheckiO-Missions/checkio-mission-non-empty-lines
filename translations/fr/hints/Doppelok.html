Vous devez combiner deux méthodes en utilisant la boucle for et la condition if. Dans la condition if, nous devons vérifier si la chaîne est vide ou non, en utilisant len(). 

text = "Hello, world!"
# Utiliser une boucle for pour itérer le texte divisé
# ("Hello," and "world!")
for i in text.split():  
    # Vérifier si les chaînes ne sont pas vides                    
    if len(i.strip()) > 0:
        # Print "OK" if all good

def non_empty_lines(text: str) -> int:

    t = 0
    for i in text.split('\n'):
        if len(i.strip()) > 0:
            t += 1
    return t

Cette option vous permet de réduire votre code et de le rendre plus léger en supprimant les variables redondantes du type t = 0. 

def non_empty_lines(text: str) -> int:
    # Ici, nous n'utilisons pas un compteur de lignes, mais une liste de lignes non vides.
    # En conséquence, nous retournons sa longueur à l'aide de len()
    return len([x for x in text.split("\n") if x.strip()])
