Ostatnie pytanie. Jak będzie wyglądać moje ostateczne rozwiązanie? 

def non_empty_lines(text: str) -> int:

    t = 0
    for i in text.split('\n'):
        if len(i.strip())> 0:
            t += 1
    return t

Proszę spojrzeć na jednolinijkowe rozwiązanie misji, ta opcja pozwala zredukować kod i uczynić go bardziej lekkim od nadmiarowych zmiennych typu t = 0. 

def non_empty_lines(text: str) -> int:
    # Here we are not using a row counter, but a list of non-empty rows.
    # And as a result, we return its length using len()
    return len([x for x in text.split("\n") if x.strip()])
