Гаразд, тобі потрібно об’єднати два методи за допомогою циклу for і умови if. В умові if потрібно перевірити чи рядок порожній чи ні, використовуючи len(). 

text = "Hello, world!"
# Використовуй цикл for для проходження
# по розділеному тексту ("Hello," і "world!")
for i in text.split():  
    # Перевір, чи рядки не пусті                    
    if len(i.strip()) > 0:
        # Надрукуй "ОК", якщо все добре
        print("OK")

Останнє запитання. Як виглядатиме моє остаточне рішення? 

def non_empty_lines(text: str) -> int:

    t = 0
    for i in text.split('\n'):
        if len(i.strip()) > 0:
            t += 1
    return t

Подивись на рішення місії в один рядок, цей варіант дозволяє зменшити твій код і зробити його більш розвантаженим від надлишкових змінних типу t = 0. 

def non_empty_lines(text: str) -> int:
    # Тут ми використовуємо не лічильник рядків, а список з непорожніх рядків.
    # А як результат повертаємо його довжину за допомогою len()
    return len([x for x in text.split("\n") if x.strip()])
