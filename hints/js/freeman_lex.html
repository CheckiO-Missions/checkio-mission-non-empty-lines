Ok, what should I start from?

Let's declare a new variable res with the initial value 0. It's your counter of non-empty lines. Your function must return its final value at the end. 

let res = 0;
...
return res

How to split the text by new line separator?

Read about split() method. Use it with new line character ("\n") as argument. 

text.split("\n")

How to iterate over separate lines of text?

Use for/of loop to iterate over lines of splitted text as values of new variable line. 

for (let line of text.split("\n")){...}

What should I do next?

Now inside loop you need to check every line for emptiness. You may do it with if statement and trim() method. It allows to remove all whitespace from the beginning and end of the string. If there is no characters in line, it become totally empty after trimming. Otherwise the line is not empty. 

if (line.trim())
    ...

If the condition is true (current line is not empty), increase the counter res. 

res += 1

So, how my final solution should look like?

You just need to wrap your code into function expression and join all previous hints together.

Here is the whole solution:

function nonEmptyLines(text: string): number {

    let res = 0;
    for (let line of text.split("\n")){
        if (line.trim())
            res += 1
    }
    return res
}

This solution is oneliner. It uses anonymous function, splits the text, filter from array of separate lines non-empty ones (has non-empty value after trimming) and take the length of the resulted array. 

var nonEmptyLines = (text: string): number => text
    .split('\n')
    .filter(s => s.trim())
    .length;

Using regular expression, find all character sequences, that end with a new line character (last word in each line) and take the length of the resulted array. 

var nonEmptyLines = text => ((text+'\n').match(/[^\n ]\n/g) || []).length
