You need to count how many non-empty lines a given text has.

Empty line is a line without symbols or with only space symbols

Input: Text.

Output: Int.

Example:

{% if interpreter.slug == "js-node" %} 
nonEmptyLines('one simple line') == 1
nonEmptyLines('') == 0
nonEmptyLines('\nonly one line\n            ') == 1
nonEmptyLines(`
Lorem ipsum dolor sit amet,

consectetur adipiscing elit
Nam odio nisi, aliquam
            `) == 3
{% else %} 
non_empty_lines('one simple line') == 1
non_empty_lines('') == 0
non_empty_lines('\nonly one line\n            ') == 1
non_empty_lines('''
Lorem ipsum dolor sit amet,

consectetur adipiscing elit
Nam odio nisi, aliquam
            ''') == 3
{% endif %}
